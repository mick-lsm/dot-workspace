# Dot Workspace

A workspace that represents data in form of nodes to boost your workflow.

# Features

**Components**

For Windows, you can directly put the .zip component file under `C:\Users\{user}\AppData\Roaming\dot_workspace\Components`

Functions

You can create  `workspace function` as developer of a component, you still can't call it now. We expect `workspace function` will work for writing script and ai agent in the workspace.
